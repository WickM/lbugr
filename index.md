Provides a high-performance 'R' interface to the 'ladybug' graph
database. Uses the 'reticulate' package to wrap the official Python
'real_ladybug' client. Enables seamless interaction with Ladybug from
within 'R' for managing database connections, executing Cypher queries,
and loading data from R data frames. Converts query results into popular
'R' data structures including 'tibble', 'igraph', 'tidygraph', and 'g6R'
objects for analysis and visualization workflows.
