## Notes

- The main difference between kuzuR and lbugr is the Python module name:
  - kuzuR uses `kuzu` Python package
  - lbugr uses `real_ladybug` Python package
- All function names are prefixed with `lb_` instead of `kuzu_`
- S3 method classes are `ladybug.query_result.QueryResult` instead of `kuzu.query_result.QueryResult`
