# Plan: lbugr - Functional Replica of kuzuR

## Goal
Make lbugr a **functional replica** of kuzuR - meaning it provides the same API/functions but for the ladybug database instead of kuzu. This is a NEW package for ladybug, not an exact copy.

**Key distinction**: lbugr uses `real_ladybug` as the Python module (not `kuzu`), and has its own version/naming. Only the function signatures and behavior should match kuzuR.
